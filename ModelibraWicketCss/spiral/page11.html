Teaching

Teaching programming courses in a business school, in a department of information systems, is a challenge. There are no that many technical courses as in a computer science department, students do not come with basic programming knowledge, and they are not motivated to learn many technical details. Since they are more interested in developing useful business applications, I had to develop a teaching strategy that would allow them to be productive as soon as possible.

Dynamic Web Applications

In my teaching, the objective is very clear, to enable students develop dynamic web applications with the minimum investment in different development technologies. I have chosen dynamic web applications since it is the most natural choice of software for business students. They are exposed to Internet in almost all business courses. Web applications are easily accessible. For business students, the fact that you can show your work to friends, relatives and potential employers is very important. The core of all information systems is a database, and dynamic web applications reflect this core in the web context.

Java

I have chosen Java and Java technologies as the principal means to develop dynamic web applications. Object oriented design and programming have been accepted by information systems practitioners. Java is an object oriented language, but simpler than other object oriented languages. Java and its related technologies have been around for quite some time and there are many available resources to help students in their learning process. For students, it is important to have a free access to technologies, and Java technologies are free of charge. In addition, there are many Open Source Software (OSS) projects that use Java. This allows me to provide easily and legally helpful tools to my students. Many OSS products have a decent, web accessible documentation that is essential for novices.

My Software

I have developed, in Java, a graphical tool for database design and code generation, a database framework for using relational databases from Java, and a web framework for constructing quickly dynamic web applications using server-side Java. Students use these tools in their course projects.

Spiral Approach

In my teaching I use the spiral approach to development. I start with some very simple examples that use a minimum of technologies, explain only concepts used in those examples, and then gradually introduce more concepts and more technologies, improving both design and development of the examples. At the beginning, students develop a few simple web pages; at the end they are able to develop useful dynamic web applications. Students also use the spiral approach to development. Week after week, they develop many versions of their applications, coming back with each spiral to already done pieces to do them in a better way, or introducing new components in a simple manner.
